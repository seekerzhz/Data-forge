CLI Usage

Common Commands

python3 forge.py workspace/example/problem.txt --provider ark
python3 forge.py workspace/example/problem.txt --num-cases 80
python3 forge.py workspace/example/problem.txt --skip-run
python3 forge.py workspace/example/problem.txt --no-generate-solution --solution-path ./solution.cpp

Key Options

  • --provider: ark | openai
  • --num-cases: total number of test files
  • --timeout: per case limit in seconds
  • --no-generate-solution: use local solution
